# ahr999-monitor

比特币 AHR999 指标监控 + WxPusher 微信推送的定时机器人。

## 运行方式

```bash
pip install requests
export WXPUSHER_APP_TOKEN=xxx
export WXPUSHER_UID=xxx
python main.py
```

定时执行由 `.github/workflows/run.yml` 驱动，每天 UTC 0:17（北京时间 8:17）自动跑一次。

## 核心文件

- `main.py` — 全部逻辑（数据获取 → 指标计算 → 条件判断 → 推送），单文件 ~160 行

## 推送逻辑一览

所有推送走 `send_wxpusher()`，未配环境变量则静默跳过。

### 底部穿越通知（边缘触发，每条仅发一次）

| 场景 | 条件 | 动作 |
|------|------|------|
| ⚠️ 跌破抄底线 | 上次 ≥0.45 → 本次 <0.45 | 推送"跌破 0.45"，仅一次 |
| 🔴 跌破深底线 | 上次 ≥0.3 → 本次 <0.3 | 推送"低于 0.3"，仅一次 |
| 🟡 回升至 0.3+ | 上次 <0.3 → 本次 ≥0.3 | 推送"回升超过 0.3"，仅一次 |
| 🟢 回升至 0.45+ | 上次 <0.45 → 本次 ≥0.45 | 推送"脱离底部区间"，仅一次 |

状态持久化到 `state.json`（记录上次 AHR999 值），每次运行后由 GitHub Actions 自动提交。

### 水平触发通知（每次满足条件都推送）

| 优先级 | 场景 | 条件 | 动作 |
|--------|------|------|------|
| 1 | ☢️ 核按钮逃顶 | AHR999≥1.5 + Pi交叉≥1.0 + FGI≥85，满足任意2项 | 推送紧急卖出，exit |
| 2 | 🔴 高估预警 | AHR999 > 1.2 | 推送停止定投建议 |
| 3 | 🚨 黑天鹅抄底 | 24h 跌幅 ≤ -15% | 推送买入 |
| — | 静默 | 其余情况 | 仅打印，不推送 |

## 数据来源

- **Binance API** — 400 天 K 线 + 24h 涨跌幅（us/com 双域名容灾）
- **alternative.me** — 恐惧贪婪指数
- **state.json** — 跨运行状态持久化（记录上次 AHR999 值用于穿越检测）

## 关键参数

```python
AHR999_BOTTOM = 0.45   # 抄底线
AHR999_DEEP = 0.3      # 深底二级预警线
AHR999_WARN = 1.2      # 高估警戒线
ESCAPE_AHR999 = 1.5    # 逃顶 AHR999 阈值
ESCAPE_FGI = 85        # 逃顶 FGI 阈值
CRASH_THRESHOLD = -15  # 暴跌阈值(%)
```
