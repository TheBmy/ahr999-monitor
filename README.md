# ahr999-monitor

一个针对 AHR999 指标的监控工具，用于辅助比特币购买决策。

## 📋 项目介绍

**ahr999-monitor** 是一个 Python 开发的监控应用，用于实时跟踪 AHR999 指标数据，帮助用户进行比特币投资决策分析。

### 什么是 AHR999 指标？

AHR999 是一个复合指标，通过结合比特币价格和 200 周移动平均线等多个因素，旨在帮助投资者识别比特币的相对价值和投资机会。

## ✨ 主要功能

- 📊 **实时监控** - 持续追踪 AHR999 指标数值
- 📈 **数据分析** - 展示指标历史趋势和当前状态
- 🔔 **告警机制** - 在关键阈值时发送通知
- 💾 **数据存储** - 保存历史数据用于后续分析

## 🚀 快速开始

### 环境要求

- Python 3.7+
- pip 包管理工具

### 安装

```bash
# 克隆仓库
git clone https://github.com/TheBmy/ahr999-monitor.git
cd ahr999-monitor

# 安装依赖
pip install -r requirements.txt
```

### 使用

```bash
# 启动监控
python main.py
```

## ⚙️ 配置

在 `config/` 目录中配置相关参数，例如：

- API 数据源
- 告警阈值
- 更新频率
- 存储路径

详细配置说明请参考 `config/` 目录下的配置文件。
